Mini Search Engine / Mini-søkemotor
===================================

## English
- Small Java project that stores words in an inverted index.
- Search supports `AND`, `OR`, and `NOT`.
- Main files: `src/InvertedIndex.java` and `src/Main.java`.
- Compile and run from `src`: `javac *.java && java Main`.

## Norsk
- Lite Java-prosjekt som lagrer ord i en invertert indeks.
- Søk støtter `AND`, `OR` og `NOT`.
- Viktige filer: `src/InvertedIndex.java` og `src/Main.java`.
- Kompiler og kjør i `src`: `javac *.java && java Main`.
